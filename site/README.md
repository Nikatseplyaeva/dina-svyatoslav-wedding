# Дина & Святослав — свадебный сайт

Статический мобильный сайт-одностраничник (HTML/CSS/JS, без сборки).
Реализован по дизайну из `project/Wedding Site.dc.html`.

## Запуск локально

Любой статический сервер подойдёт, например:

```sh
cd site
python3 -m http.server 8000
```

Открыть `http://localhost:8000`.

## Структура

```
site/
  index.html          — разметка всех секций
  css/styles.css       — все стили
  js/petals.js          — анимация падающих лепестков (canvas)
  js/rsvp.js             — логика формы RSVP + отправка ответа
  js/config.js            — токен и chat ID Telegram-бота (см. telegram-bot/README.md)
  assets/art/            — иллюстрации пары
  assets/tex/              — текстуры васи (фон секций)
  telegram-bot/            — инструкция по настройке бота для приёма RSVP
  apps-script/             — старый вариант через Google Sheet (не используется,
                              оставлен на случай, если понадобится вернуться)
```

## RSVP-форма

Форма отправляет каждый ответ сообщением в Telegram через бота.
Перед публикацией сайта нужно один раз настроить бота — инструкция в
[`telegram-bot/README.md`](./telegram-bot/README.md), затем вставить
токен и chat ID в `js/config.js`.

(Раньше форма писала в Google Sheet через Apps Script — код остался в
`apps-script/`, но отключён: у части гостей запросы к
`script.google.com` не проходили из-за сетевых ограничений, при этом сайт
не мог отличить это от настоящего успеха.)

## Деплой

Это чисто статические файлы — можно захостить на Netlify, Vercel (как
static), GitHub Pages, Cloudflare Pages и т.д. Просто указать `site/` как
корень публикации.
